# finance
Harvard CS50 pset8

A python web application using flask designed to mimic a marketplace for buying and selling stocks.
<br>
Some features include:
<ul>
  <li> Register as new user </li>
  <li> Log in page </li>
  <li> Change password </li>
  <li> Get a stock quote using real time prices </li>
  <li> Buy and sell shares </li>
  <li> Deposit more cash </li>
  <li> View transaction history </li>
</ul><br>
An SQLite database is used to store the relevent user information.
